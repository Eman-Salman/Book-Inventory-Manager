# Book Inventory Manager

A simple WPF desktop application built in C# using Visual Studio 2017.

## Features
- Add, Update, Delete books
- Display book list in DataGrid
- Save book data to CSV file on Desktop
- Count total books
- Clean UI with event-driven functionality

## OOP Structure
- `Book` class with properties: `Id`, `Title`, `Author`, `Quantity`.
- Encapsulation using properties
- Constructors used for initialization

## Data Storage
- CSV file stored on Desktop: `BookInventory.csv`

## How to Run
1. Open the solution in Visual Studio 2017
2. Build the project
3. Run the app

## GitHub
1. Create a new GitHub repository
2. Push the solution folder to GitHub
